Some features that make PlaceMaker awesome

  • Location Enabled

    PlaceMaker uses iPhone location to determine the nearest stops and display travel time.

  • Flexible Search

    Search using either Google or Apple Maps. If you can't find a place in one, use the other.

  • Choose Your Navigation App

    Use navigation with your favorite navigation app. PlaceMaker supports Waze, Google Maps and Apple Maps.

  • Custom Order

    Save any number of stops. Sort them manually, in the order you want, or sort them by distance.
